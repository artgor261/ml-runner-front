GET /health

# Datasets

## Загрузка данных с MOEX

POST /datasets/moex

Создаёт новый датасет на основе данных Московской биржи.

Поддерживается загрузка нескольких тикеров одновременно.

---

## Импорт локального датасета

POST /datasets/local

Регистрирует каталог с parquet-файлами как датасет.

---

## Импорт из Google Drive

POST /datasets/gdrive

Импортирует датасет по ссылке или ID Google Drive.

---

## Список датасетов

GET /datasets

Возвращает список всех зарегистрированных датасетов.

---

## Получить датасет

GET /datasets/{dataset_id}

Возвращает подробную информацию о датасете.

---

## Удалить датасет

DELETE /datasets/{dataset_id}

Удаляет датасет.

---

# Training

## Запуск обучения

POST /training/runs

Создаёт новый запуск обучения.

Поддерживает передачу гиперпараметров напрямую.

---

## Запуск обучения из JSON

POST /training/runs/from-file

Создаёт новый запуск обучения на основе JSON-конфигурации.

---

## Список запусков

GET /training/runs

Фильтрация:

* active=true
* active=false
* status

---

## Получить запуск

GET /training/runs/{run_id}

Возвращает краткую информацию о запуске.

---

## Статус запуска

GET /training/runs/{run_id}/status

Возвращает:

* статус;
* текущую эпоху;
* историю loss;
* текущие метрики.

Используется для мониторинга обучения.

---

## Остановить запуск

POST /training/runs/{run_id}/cancel

Останавливает обучение.

---

# Experiments

## Создать эксперимент

POST /experiments

Создаёт новый эксперимент.

---

## Список экспериментов

GET /experiments

Возвращает все эксперименты.

---

## Получить эксперимент

GET /experiments/{experiment_id}

Возвращает полную информацию об эксперименте.

---

## Запуски эксперимента

GET /experiments/{experiment_id}/runs

Возвращает список запусков эксперимента.

---

## Все запуски

GET /experiments/runs

Плоский список всех запусков.

Аналог страницы Runs в MLflow.

---

## Детальная информация о запуске

GET /experiments/runs/{run_id}

Возвращает:

* гиперпараметры;
* историю обучения;
* метрики;
* пути к артефактам;
* ошибки выполнения.

# Models

---

## Зарегистрировать существующую модель

---

## Список моделей

GET /models

Возвращает все зарегистрированные модели.

---

## Получить модель

GET /models/{model_id}

Возвращает подробную информацию о модели.

---

## Удалить модель

DELETE /models/{model_id}

Удаляет модель из реестра.

---

# Validation

## Запуск валидации

POST /validation/run

Выполняет проверку модели на тестовых данных.

Источники данных:

* зарегистрированный датасет;
* каталог parquet-файлов.

Источники модели:

* зарегистрированная модель;
* путь к .pt файлу.

---

## Результат валидации

Возвращает:

### Metrics

Словарь рассчитанных метрик.

### Predictions

Для каждого тикера:

* реальные значения;
* предсказанные значения;
* индексы временного ряда.

### Date Ranges

Временные диапазоны для построения графиков.

### Backtest

Результаты бэктеста, если он был запрошен.

# Статусы запусков

pending

Ожидает запуска.

running

Выполняется.

completed

Успешно завершён.

failed

Завершён с ошибкой.

stopped

Остановлен пользователем.
